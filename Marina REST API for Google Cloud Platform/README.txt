Project Name: Marina REST API for Google Cloud Platform
Author: Casey Cheek
Date: 12/3/2021
Class: CS493 - Cloud Application Development

Project Description: 
This is a RESTful API that models a simple marina business, managing information about three entities: Users, Boats, and Loads.  It is written in Python with Flask, built to be deployed on Google Cloud Platform, and uses Auth0 to verify its users.  This application supports full CRUD operations for each entity and includes an extensive Postman test suite.  Make sure to read the API Spec pdf for more details.
